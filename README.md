# Future Imóveis — Condomínio Marbello

Landing page de captação de leads para venda de lotes do Condomínio Marbello, em Paraipaba-CE.

O contexto de negócio, a estrutura do código e o local de cada informação editável estão no [guia do projeto](docs/PROJECT_GUIDE.md).

## Desenvolvimento

```bash
npm install
npm run dev
```

## Validação

```bash
npm run typecheck
npm run build
```
